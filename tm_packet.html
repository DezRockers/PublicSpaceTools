TM Packet

This tool displays the individual fields of the input TM PUS packet in hexadecimal bytes.

Input the TM string as either non-delimited hexadecimal bytes and then click 'Add Spaces to TM String' button
or space delimited hexadecimal bytes and then click 'Display TM Fields' button

Input the length of the time field in bytes. This is mission specific and is usually 7 bytes long. Default is 7.
Input the length of the spare field in bytes. This is also mission specific. Default is 0.
